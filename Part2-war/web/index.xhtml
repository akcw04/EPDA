Enterprise Programming - Dynamic Appointment System

System Overview

Architecture: 3-Tier MVC (JSF/PrimeFaces - EJB - JDBC)

  • View Layer: JSF with PrimeFaces components
  • Business Logic: Jakarta EE Stateless EJBs
  • Data Access: Raw JDBC with try-with-resources

Key Features

  • Appointment management with technician availability validation
  • Service types: Normal (1 hour) and Major (3 hours)
  • Customer management and feedback system
  • Technician workload tracking
  • Comprehensive reporting dashboard
  • SHA-256 password hashing in business tier
